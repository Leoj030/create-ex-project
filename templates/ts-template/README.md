# Getting Started

## This is your Express project using Javascript

### To get started, you can run the following commands:

#### To run the project in development mode (Auto restart on file changes)

```bash
npm run dev
```

#### To run all the test files in the test folder (Works only if you have test files)

```bash
npm test
```

#### To build the project

```bash
npm run build
```

#### To start the production project

```bash
npm start
```

---

### Recommended Packages to install

- mongoose
  - For easy access to MongoDB with schema.

- express-validator
  - For easy validation of request data.

- jsonwebtoken
  - For easy generation of a token for authentication.

- bcryptjs
  - For easy hashing of your password.

- uuid
  - For easy generation of a unique id for your user.
